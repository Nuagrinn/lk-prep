# Лайвкодинг: практические задачи

Условие, сигнатура функции и примеры с ожидаемым выводом лежат прямо в
`.go`-файле задачи как doc-комментарий, вместе с `main()`, который сразу
запускает оба примера и печатает `OK`/`MISMATCH`. Пиши решение прямо в этом
файле и запускай `go run <файл>.go` - результат виден сразу, без ручной
сверки вывода.

## Как сдать решение

Допиши `TopKFrequentWords` в файле задачи и запусти его. Когда оба примера
печатают `OK` - можно считать задачу решённой, статус в таблице выше
обновляется на "Решено".
